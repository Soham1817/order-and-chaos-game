# Order and Chaos Game

Order and Chaos is a strategic two-player game played on a 6√ó6 grid where players alternate placing Xs and Os. One player aims to create a straight line of five matching symbols (Order), while the other tries to prevent it (Chaos).

## How to Use

1. **Choose your symbol** (`X` or `O`) before making move.
2. **Click on a cell** in the 6√ó6 grid to place your symbol.
3. The game tracks rounds, updates scores, and declares a winner automatically.
